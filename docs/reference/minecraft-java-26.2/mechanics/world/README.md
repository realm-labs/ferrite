# World mechanics

[Back to the leaf-rule manual](../README.md).

Each file contains one implementation-level leaf rule. Stable rule IDs remain the normative
references used by behavior pages, the completion ledger, the catalog, and tests.

## Leaf rules

### [`WGEN-PIPELINE-001`](wgen-pipeline-001.md)

Chunk generation advances status-by-status with locked data inputs

### [`WGEN-STRUCTURE-BURIED-001`](wgen-structure-buried-001.md)

Buried treasure descends one anchored piece and encloses a loot chest

### [`WGEN-STRUCTURE-NETHER-FOSSIL-001`](wgen-structure-nether-fossil-001.md)

Nether fossils scan for a cavity, place one bone template, and may add a dried ghast

### [`WGEN-STRUCTURE-IGLOO-001`](wgen-structure-igloo-001.md)

Igloos terrain-anchor a top and an optional laboratory shaft

### [`WGEN-STRUCTURE-SWAMP-HUT-001`](wgen-structure-swamp-hut-001.md)

Swamp huts terrain-average a fixed cabin and latch two occupants

### [`WGEN-STRUCTURE-DESERT-PYRAMID-001`](wgen-structure-desert-pyramid-001.md)

Desert pyramids span four chunks and make one global archaeology selection

### [`WGEN-STRUCTURE-JUNGLE-TEMPLE-001`](wgen-structure-jungle-temple-001.md)

Jungle temples randomize masonry and persist two traps and two chests

### [`WGEN-STRUCTURE-SHIPWRECK-001`](wgen-structure-shipwreck-001.md)

Shipwrecks defer live height, choose one of eight palettes, and seed marker chests twice

### [`WGEN-STRUCTURE-RUINED-PORTAL-001`](wgen-structure-ruined-portal-001.md)

Ruined portals choose one center-owned template transaction and grow an unclipped netherrack apron

### [`WGEN-STRUCTURE-OCEAN-RUIN-001`](wgen-structure-ocean-ruin-001.md)

Ocean ruins restack live-height templates, globally cap archaeology, and spawn marker drowned

### [`WGEN-STRUCTURE-STRONGHOLD-001`](wgen-structure-stronghold-001.md)

Strongholds retry a weighted piece graph until one portal room exists

### [`WGEN-STRUCTURE-MINESHAFT-001`](wgen-structure-mineshaft-001.md)

Mineshafts grow a depth-first wood graph and choose support, carts, and spiders from live chunks

### [`WGEN-STRUCTURE-END-CITY-001`](wgen-structure-end-city-001.md)

End cities transact recursive template groups and share a process-global ship latch

### [`WGEN-STRUCTURE-FORTRESS-001`](wgen-structure-fortress-001.md)

Nether fortresses alternate quota-weighted bridge and castle frontiers

### [`WGEN-STRUCTURE-OCEAN-MONUMENT-001`](wgen-structure-ocean-monument-001.md)

Ocean monuments prune a connected room lattice inside a fixed flooded shell

### [`WGEN-STRUCTURE-WOODLAND-MANSION-001`](wgen-structure-woodland-mansion-001.md)

Woodland mansions tile a randomized floor graph into saved templates

### [`WGEN-JIGSAW-CORE-001`](wgen-jigsaw-core-001.md)

Jigsaw placement expands prioritized pool connectors through a bounded collision volume

### [`WGEN-JIGSAW-RECORDS-001`](wgen-jigsaw-records-001.md)

Ten locked records parameterize the generic jigsaw transaction

### [`WGEN-JIGSAW-PROCESSORS-001`](wgen-jigsaw-processors-001.md)

Structure processors form an ordered, position-stable rewrite transaction

### [`WGEN-JIGSAW-ANCIENT-CITY-001`](wgen-jigsaw-ancient-city-001.md)

Ancient-city pools compose sparse redstone ruins and fixed container payloads

### [`WGEN-JIGSAW-BASTION-001`](wgen-jigsaw-bastion-001.md)

Bastion pools compose destructive air masks, loot rooms, spawners, and finalized mobs

### [`WGEN-JIGSAW-OUTPOST-001`](wgen-jigsaw-outpost-001.md)

### [`WGEN-JIGSAW-TRAIL-RUINS-001`](wgen-jigsaw-trail-ruins-001.md)

Trail ruins age ordinary templates before globally capped archaeology

Trial-chamber pools bind one structure-wide mob roster to destructive copper rooms

### [`WGEN-JIGSAW-VILLAGES-001`](wgen-jigsaw-villages-001.md)
